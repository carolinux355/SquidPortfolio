# SquidPortfolio
This is a collection of various player-facing features I implemented for Jellyvale. Some of the features I implemented or took over ownership of include:
- Town expansions, fog boundaries, and playable tiles
- Quests: including main storyline, daily generated asks, and long term achievements
- Narrative: data, UI, tooling, and animation support
- Town character animations and behaviors
- Lots of iteration and implementation for our new user onboarding, for both new installs on day 0 and introducing new features as they get unlocked
- Daily login gifts and task list
- Player Collections (fireflies, rare books, plushies, etc)
- Rose Garden feature (complete special levels to rescue bees and send them to pollinate magical roses) including core loop, animation work, and progression map
- Daily shop of rotating goods selected randomly from a pool of eligible items
- Name Your Town: concepting the feature and implementing, intended to boost D0 retention
- Farming: early prototyping of various models playtested internally, then taking it to production quality, owning the engineering work and managing the feature as a whole, gathering feedback from playtests and assigning action items to teammates, iterating on economy tuning and finalizing polish
- Implementing A/B tests for various features in development and closing out experiments once winning variants were determined. Tests ranged in complexity from basic tuning swaps to fully reimplementing the early game experience.
